KitIcon is icon block  for WordPress Gutenberg Block.

== Description ==
WordPress gutenberg icon blocks or WordPress gutenberg block icon. This is icon Creator allows you to build custom design and gutenberg widget your content..
== Here is 500+ icon and 500+ icon is coming soon ==
To use drag the "gutenberg blocks" blocks from the "KitIcon" section of the add block + icon sidebar onto you post or page. You'll see a preview of what things will look like in the main content panel.

== Installation ==
Make sure gutenberg is activated, then activate this plugin. You'll see the block under the "KitIcon" section in the add block section.

== Frequently Asked Questions ==

= Can I use the plugin without gutenberg ? =

No. You cannot use without gutenberg editor since it's an addon for gutenberg editor.

= Does it work with any theme? =

Absolutely! It will work with any theme where gutenberg editor works.

= Will this plugin slow down my website speed?  =

No, the plugin don't slow your website.
